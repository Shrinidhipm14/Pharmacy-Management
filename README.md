# 💊 Pharmacy Management System

## About the Project

This project focuses on developing a Pharmacy Management System to manage medicine inventory, customer details, and pharmacy operations efficiently. The system helps in maintaining medicine records, tracking stock availability, and simplifying day-to-day pharmacy management activities.

## Objectives

* Manage medicine details and inventory records.
* Maintain customer and sales information.
* Track medicine availability and stock levels.
* Reduce manual work in pharmacy operations.
* Provide an efficient system for managing pharmacy data.

## Features

* Medicine inventory management.
* Add, update, and delete medicine records.
* Search medicines based on different attributes.
* Manage customer and sales details.
* Maintain organized pharmacy records.
